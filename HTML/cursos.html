Curso Técnico em Desenvolvimento de Sistemas

Objetivos do Curso

Capacitar os alunos a desenvolverem soluções de software de alta qualidade, utilizando as melhores práticas de programação e desenvolvimento ágil. O curso prepara profissionais para desenvolver sistemas, sites e aplicações. Aborda algoritmos, programação, banco de dados e práticas de engenharia de software.

Disciplinas

  • Algoritmos
  • Programação Java
  • Banco de Dados
  • Desenvolvimento Web
  • Engenharia de Software

Duração: 2 anos

Modalidade: Presencial

Curso Técnico em Cybersegurança

A área de Cybersegurança (ou Segurança da Informação) é uma das mais críticas e bem pagas da tecnologia atualmente. O curso combina fundamentos de redes, programação, direito digital e psicologia (engenharia social). Diferente de um curso de TI comum, aqui o foco é proteção, detecção e resposta. Abaixo, detalho as principais disciplinas divididas por eixos temáticos.

Disciplinas

  • Fundamentos de Redes
  • Programação para Segurança
  • Direito Digital
  • Psicologia e Engenharia Social
  • Resposta a Incidentes
